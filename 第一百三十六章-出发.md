# 第一百三十六章：出发

苍穹纪元2170年8月23日。星海号从月球轨道站启航。

林小溪站在飞船的观景舱里——看着地球越来越小——直到它变成一颗蓝色的星点——淹没在无数其他星点之中。

"第一次离开太阳系——什么感觉？"同行的乘客问。

"不是离开。"林小溪说，"是出发。"

飞船的星门引擎启动了。周遭变得模糊——时间在这里失去了意义——然后——她看到了另一个太阳。

信使文明的母星——在四光年之外——以一秒钟的速度出现在她的视野中。

那是一个蓝色的太阳——不是地球太阳那种黄色——是更深邃、更宁静的蓝色。在蓝色太阳的周围——十二颗行星按照精确的轨道运行。其中一颗——被厚厚的云层覆盖——看起来像一颗蓝色的珍珠。

"欢迎来到信使文明的家园。"飞船的AI说，"那是我们的母星——数据之海。"
