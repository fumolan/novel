# 第一百一十二章：航线图

## 一

林小溪的深空导航系统在苍穹-Σ共轭层上线后，进入了一个全新的阶段。

过去五年——按照地球时间算——她在深空研究所积累了大量数据。逐日号穿越银河系中心时的航线记录、信使文明的往返轨迹、联盟探测船的通信响应延迟——所有这些数据都通过苍穹-Σ的观测系统被结合到了一起。

只有一个问题：数据不完整。

"人类只有一条已探索的深空航线。"她在A3区导航实验室的白板上画了一条弯曲的线，从太阳系出发，绕过银河系中心，到达档案馆，"逐日号走的这条线是极端条件下的应急航线，穿过了高辐射区，绕道三个无法停靠的星际区域。它不是一条可重复使用的航线——它是一次性的。"

"我们需要什么？"陆辰问。

"一张完整的星际航线图。从地球出发，通往联盟七个成员文明各自的主星，经过至少十二个安全的恒星系，覆盖现有的所有通信中继站——包括Dr. 方在小行星带的那个。"

"这是一整个导航网络。"

"对。但现在我们只有一张草图——逐日号的实际航线数据。"林小溪在屏幕上调出那张航线草图，它看起来像一条弯曲凌乱的涂鸦线，"这张图的局限性很明显：它只有一条线——而且是紧急情况下走的灰区航线。一张完整的星际航线图需要从七个文明的导航数据中提取信息，在联盟的统一参考系下整合，最终产生一张多层的地图：安全航线层、通信中继站层、应急备选路线层。"

"联盟有没有现成的航线图？"

"有。"林小溪打开一张联盟的共享航图——它看起来非常完整，七条彩色的航线从银河系的各个方向汇聚到联盟中央，"但你不能直接用——因为联盟的航图坐标系是克瑞尔文明定义的，它有一个隐含的假设：所有导航飞船都配备了克瑞尔标准的惯性阻尼器。"

"逐日号没有克瑞尔阻尼器。"

"逐日号没有。所以如果在联盟的坐标系下计算逐日号的航向，会有一致性误差。这个误差按百分比算不大——但在星际导航中，百分之零点一的误差，意味着在跨越数十光年时，你可能会偏出安全航道。"

陆辰走前去，看着那张联盟的共享航图。

"你需要做的是——在联盟坐标系上叠加逐日号的实际航迹，消除阻尼器偏差，然后把修正后的地图回传给联盟。"

"不止。我还要标注出所有人类飞船适用的安全停靠点。"林小溪的手指在屏幕上滑动，"信使文明不需要停靠——他们是数字意识。但人类是物质的，人类的飞船需要重力、空气、水——在联盟的标准航线上，适合人类停靠的站点目前只有三个。"

"联盟能帮忙扩展停靠点吗？"

"能。但前提是我们先拿出一张完整的航线需求图。"她看着陆辰，眼睛里有一种他已经很熟悉的光——不是在征求许可，而是在告知决定，"我需要逐日号再飞一次。不是去找档案馆——是去测绘航线。"

"第二次深空飞行。"陆辰低沉地说。

"逐日号2.0。从地球出发，沿联盟主航道飞行，在十二个恒星系设置导航信标，建立人类的第一批星际停靠站。"

"你要做多长时间？"

林小溪沉默了一会儿。然后她调出一个预估。

"往返时间——不算时间膨胀——大约四年。"

大厅里安静了几秒。陆辰看着那张预估——四年。对地球上的人来说是四年，对飞船上的人来说是四年。这一次没有黑洞引力场，没有时间膨胀，四年就是四年。

"我去。"林小溪说。她的声音很平稳，但陆辰能听出她隐藏的情绪。

"我知道。"

"我在离开之前，需要你帮我做两件事。"

"你说。"

"第一——把苍穹-Σ的地面导航模块扩展到星际规模。逐日号2.0的导航系统需要能实时接入苍穹-Σ的观测数据，反向计算地球和太阳系的状态变化。这是导航的基础——知道你在哪里，需要先知道'哪里'在哪里。"

"底层架构扩展。"陆辰在脑子里开始推算工作量，"从地球尺度到银河尺度——需要重新定义空间坐标的精度范围。"

"第二——"

她停了一下。陆辰有一种熟悉的感觉——好像回到了多年前，逐日号第一次出航的那个晚上。

"等我回来。然后我们——一起去看那个坐标。你父亲和Dr. 方去过的——终点的那个位置。"

陆辰看着她。这一次不像五年前——五年前她离开时，他不知道自己还能不能见到她。现在他知道了一件事：她会安全回来的。不是因为概率——是因为她的能力配得上那条航线。

"四年。"他说。

"我会发数据回来。每天。苍穹-Σ的观测信道足够宽——不管我在多远的地方，你都能收到。"

"我会看着。"

林小溪点了点头，转身走了。走到门口时，她停了下。

"还有一件事。"

"什么？"

"我给航线图取了个名字。"

"什么名字？"

"'星桥'。"她说，"不是桥接两个点。是桥接一个文明和自己从未见过的星空。"

## 二

星桥航线图的测绘工作从六月初开始。

这不是一场飞行——飞行要到逐日号2.0建成后才能开始。这之前有大量的准备工作。林小溪和她的团队花了两个月时间建立了一个完整的航线模拟系统，用联盟提供的多文明导航数据作为基础，叠加人类飞船的工程参数，生成了一份预研版的航线图。

预研航线图发布那天，深空研究所把整个A3区变成了一个沉浸式的三维导航厅。穹顶上是银河系的缩小投影，航线像发光的丝线在星系间游走。十二个标注的恒星系像沿途的珍珠一样依次亮起——从太阳系出发，经过半人马座比邻星、克瑞尔主星旁道、信使文明的虚拟坐标、维拉主星——每一个经停点都有详细的资源数据和风险评估。

"维度有多大？"陆辰站在导航厅中央，仰头看着穹顶。

"从地球到最远的经停点——大约四百二十光年。"林小溪站在他旁边，"这是联盟主航道中人类飞船在当前技术条件下能安全穿越的最大范围。在更外围——你看到的那些灰线——是'未验证航线'。在联盟的旧航图中它们被标记为不可靠——但有可能，在未来，人类验证了其中一些。"

"这些灰线会变成新的航线。"

"对。但前提是我们的技术继续进步。"

陆辰看着那些灰色的虚线。它们在银河系的外围延伸，指向更远的星域——有方向但没有坐标，有意向但没有实际的经停点。他知道那些线上有什么——更多他从未见过的文明，更多的评估中、等待认证的文明。那些灰线是人类向外的方向。

"航线图什么时候公布？"他问。

"两个月后——"林小溪看了一眼日期，"逐日号2.0首次飞行测试之日。我想在那一天公布航线图。"

"首发航线数据库，全面公开？"

"是的。任何研究机构、任何飞船的导航系统、任何一个对人类有兴趣的联盟文明——都可以下载星桥航线的完整数据。"

"这不算机密吗？"

"韩墨白问过同样的问题。"林小溪微微一笑，"我的回答是——航线图不是武器。航线图是信息的共享。宇宙已经足够大了——航线不会让敌人找到我们，只会让朋友更快地到来。"

陆辰看着她。她站在银河系的三维投影中，脸上被星光映出一种柔和的蓝色。五年前她是那个在B7层紧张地追问他堆内存增长率的女孩，现在她在为整个文明画一张通往外界的路由表。

"还有一个事。"林小溪说，"你有没有注意到——这些经停点中有一个是重复出现的？"

"哪个？"

林小溪放大了一段航线——在维拉主星和小行星带中继站之间，有一个跳转节点。这个节点在联盟的所有旧航图中都没有出现过，但在逐日号上一次飞行的数据中——以及Dr. 方留下的中继站的传输日志中——都有标记。

"这个节点的坐标指向一个空的空域——那里没有任何已知的恒星或行星。但Dr. 方的数据传输路径经过了它——而且逐日号在一次偏离航道时，被动扫描到了一组不应该出现的光谱信号。"

"什么类型的信号？"

"光信号非常微弱——但在解析校正之后，它的调制波型与Dr. 方中继站上那个未识别过滤器的波形——一致。"

林小溪的声音压低了几分，"那条航线上有一个我们还没看到的东西。在'星桥'的第二次飞行中，我要弄清楚它是什么。"

陆辰看着那个灰点。它很小，在航线图上不如那些标注过的恒星引人注目。但他有一种感觉——这个灰点可能比他们现在所知道的一切都更重要。

## 三

七月十九日，逐日号2.0在月球轨道站完成了最后的组装。

这艘新船比第一代大了三倍，搭载了最新的深空引擎和基于苍穹-Σ导航架构的自动驾驶系统。她看起来不像一艘探索船——她看起来像一扇移动的门。

林小溪在起航前给陆辰发了一条简短的消息。

"出发了。每天发数据回来。四年后见。"

"好的。"

就两个字。但林小溪知道那两个字意味着什么。

升空的那一刻，月球轨道站的观测舱里站着很多人。苏晚晴、韩墨白、赵恒、陈默——还有方如清。他们都来送她。但陆辰没有来。

他坐在北京数据中心B7层的工位上，"深海"的界面上，逐日号2.0的导航系统的实时状态正在跳动。绿色的光网中，一条发光的细线从月球轨道延伸出去，向银河系的深处伸展——那是逐日号的航迹。

星桥航线图的第一条线，正在被画出来。

陆辰的手指在键盘上轻轻敲下一个注释。

```
// 航线 №1 · 方向：向外的方向
// 测绘者：林小溪
// 起航时间：苍穹纪元2162年7月19日
// 预计返回：苍穹纪元2166年7月
```

他保存了注释，关掉了终端。

窗外的天空正在变暗，但苍穹的防护网在蓝光中柔和地闪烁。那层光膜现在像一层透明的面纱——不是屏障，不是笼子，只是一个提醒。提醒地球上的所有人——在这层光膜的上面，有一片广阔的、没有边界的星穹。

他去食堂吃了一份简单的晚餐。米饭、青菜、一碗紫菜汤。他一个人坐在角落里吃着。

他想到了林小溪在航线图中标注的那个灰点。

想到了Dr. 方中继站上的那个未识别的秘密。

想到了父亲在终点站最后说的一句话——

它很美。

然后他想到了再过四年，他会站在月球轨道站上，等着逐日号2.0从航线末端回来——

等着林小溪穿过那扇门。
