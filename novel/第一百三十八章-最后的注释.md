# 第一百三十八章：最后的注释

## 一

苍穹纪元2199年最后的几天。

陆辰做了一个他一直在推延的决定——正式移交苍穹-Σ首席架构师的职位。

他不是离开——他仍然是系统的最底层维护者之一。但他把首席位置交给了一个年纪比他轻二十岁的双栖工程师——半人类血统，一年已在月球和地球之间形成了正常的职业跨度。这个年轻人叫杨舟——是星星的表哥，在B7层工作了七年，用"深海"写过三层类加载的各型自定义载入，并在共轭层明辨安全栈方面受过苏晚晴亲手记述。

陆辰在移交文件上只写了一句话：

> 首席架构师不是所有者。当系统不再需要一个人来决定所有核心——而成了许多人维护的共有地——这个职位就不再需要我。

移交完成后，陆辰从B7层的工位上清出了所有的个人物品——书籍、日志、那颗早已褪色的火魄碎片——装进一个旧帆布包。

赵恒在茶室替他留了一杯咖啡。他喝了最后一口——然后环顾这一个他坐了四十年的地下室。

四十年。那排监控屏幕依然像四十年前一样闪烁。

"你以后还会来吗？"陈默问。

"也许——每周一次。来看看日志。来喝赵恒的咖啡。来跟星星讨论她的航线图。"

"你不必走——你只要留在北京——这里就是你的工位。"

"我知道。"陆辰把包拎上肩，"但我不想让自己成为系统中不可替代的人。或者更别说——我老了。我已经经不起凌晨三点被别人电话叫醒。我的时间——应该给那些能像我当年一样伸出手的年轻人。"

他走到电梯口。赵恒、陈默、茶室里新一批的底层工程师——都站起来看着他。

"这是B7层。不是告别的地方。"陆辰说，"我明天早上八点——还会回来的——去买自动售货机里的咖啡。你们的咖啡机太贵。"

电梯门关上。一个时代悄然翻页。

## 二

那晚——在苍穹纪元2199年12月31日的最后一个小时。陆辰回到了B7层——不是以首席身份——只是以陆辰的身份。

他坐在他的老工位上，打开"深海"，接入苍穹-Σ核心协议引擎。

他的火珀级权限依然有效——这份权限不与职位挂钩，只跟人。他从来没用它做过任何与职位有关的事——现在他需要最后一次用它。

在苍穹-Σ的Bootstrap ClassLoader启动序列中——那段Dr. 方留下的原始注释"如果有一天有人看到这行注释——"已经存在了超过半个世纪。在它下面——陆辰在二十多年前加了一条——"苍穹曾经是屏障。但最终——它会成为一扇门。"

现在——他在这两行注释之下——写下第三行。

他的手指在键盘上安静地敲下：

```
// 苍穹-Σ · 第三行
// 门不会关闭。门已经种在代码里——任何读懂这些注释的人——都能打开它。
// 如果你看到这里——请记得上一代人是为你在门外画出了路标——包括父亲和Dr. 方。
// 不要守着门——帮别人走出去。
//
// ——陆辰 · 苍穹纪元2199年12月31日
```

他盯着屏幕看了很久。然后他按下保存，退出编辑。

在苍穹纪元2200年的零点——新的元年钟声敲响之前——他把"深海"关掉了。

陆辰站起来，拎起他的旧帆布包，走出了B7层。

他没有回头。因为他知道——这扇门不会再关上。代码已经种好了——无论是他，是陈默，是杨舟，是星星——都已经学会怎么打开它。

## 三

陆辰走出北京数据中心的那一刻——

苍穹纪元2200年零点。

整个城市的天空被烟花点亮了。明亮的光——交替闪烁在那层淡蓝的保护膜之上——那层膜现在几乎完全透明——像是天空上未经估价的一层温暖光晕。

他的口袋里现在有着那颗已经完全不发光的火魄碎片。他取出它，在手心轻轻握着。

"爸——Dr. 方——我们到了。"

他收回视线，向隔壁的那家羊肉汤店走去——小张对他说过——新年的第一碗汤免费。

刚走到门口——他看到林小溪已经在店里靠窗的位置坐着——桌上两碗热汤。她抬起头，又转过来，像等着他在扩展墙刚褪去的烟雾中走近。

她指了指她旁边椅子上的包："你搬家也得喝——这碗里面放了双份山药——小张专门给你盛的。"

陆辰在她对面坐下，拿起勺子。

窗外烟花继续。门边还有别的食客——老的年轻的一起在人造蓝光下吃跨年饭。

他对林小溪说："时间还在——明天见。"

她把汤勺递给他——两个人的手腕在碗边轻触了一下——不是人的关节声——是一小段频率——她在航线上听着地面、他在地面校正航——四十年加密通信——延续在一个羊肉汤店里。

她抿了一口汤。然后轻声说：

"新年。"

陆辰点了点头。店门外的黑搞定在了——苍穹控光以千年算——刚又更换了新年段。

在这座城市的任意一个掩护层下——宇宙在继续。
