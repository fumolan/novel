# 第3章：浇筑工小码

## 一

老人说，他想尽快住进去。

陆终在纸上画了一稿草图——不是蓝图晶体的全息投影，可以用旧纸和笔徒手画的那种。老人看不懂建筑的规范说明，但陆终画了一个长方块的轮廓。一个小方块代表向东的窗户。一条斜线代表日出时可能落在墙上的光。

但那天他还没回到终点事务所，小码就已经动手了。

"你在干什么？"陆终站在自己工作室门口，声音是那种压到最低、反而更危险的低沉。

小码蹲在一个刚刚挖开的浅坑旁——就在终点事务所侧面原本空置的狭长地块上。他手里握着一根粗钢筋，脚下是一摊已经混合好的浇筑料——配比不准，稀了，能看到表层浮着过量的水分。旁边地面上线已经画好了，两个十字交叉作为定位标记——但是比例失调了，意味着长方体短边位置设计里压根没对。一堵结构粗矮的短墙沿着标线的其中一条边冒了出来，比设计图上的基础定位垂直偏移了至少三十厘米。歪的。

"你不是说要给老头盖房子吗？"小码头也不抬，钢筋继续往下捣，"我先把地基的口感丰一丰——"

"我没给你图纸。"

"我目测了！你那抽屉里那些旧方案图，我翻过一张类似的。"

"那张墙厚零点三，这面墙——"陆终蹲下来，手掌搭在浇筑面的截面，"你浇了快零点六。既然都不是按图施工，跟我说一声"我目测了"——恕我一贯没耐心被这个开头展开。"

小码终于停手。他站起来——比陆终矮半个头。衣服前襟沾满了半干的水泥点渍。他把手往裤子上擦了擦——没擦干净。

"可是你看——"小码指着那堵歪墙,"它虽然歪了，但能挡风。"

陆终看着那堵墙。墙的位置比标准地基转角偏了将近半米，厚度超标——但确实屹立不倒。

他没有说话。沉默持续了十几秒，以至于小码开始挪脚。

但陆终不是在生气。而是：这面位置错误的、倾斜的、超厚的挡土短墙——歪打正着地挡住了从废弃管道方向吹过来的穿堂风。那个管道口是他之前设计时没注意到的风源——它会导致冬天室内温度分布不均。但在标准流程中，他会在施工图上标注"增设挡风结构"并在第二步执行时处理。小码直接从第一步跳到了错误的第二步——却把第三步的关键问题顺手先解决了一半。

"你刚才浇的时候——脑子里想的是什么？"

小码愣了一下，挠了挠后脑——更多的水泥屑从发梢落到他肩膀上："没特别想什么啊。就是觉得那边应该有堵墙。手比脑子快。"

陆终看着他。不是审视。是调试。

一个能跳出标准流程、能在没有完整信息时做出判断、能造出一个歪但结构上能用的东西的人——这种人不能被按回固定流程里去使用。

他拿出笔，在草图上把转角标记改了。然后在小码的肩膀上拍了一下。

三十厘米的偏移校正了过来。

"这次我改设计图——下次，你先等我看完你的判断再说。"

小码本来准备好了挨骂，没想到听到的是这个："……啥？你不骂我乱浇？"

"歪了怎么做，不改原先的基础你就把墙扶在上面焊——我把设计改成朝那个方向倾斜零点三度，利用结构补偿。以后你浇之前，先在地上画线。画完了我来看，再动手。"

"你信我画线？"

"我看过你画的线——"陆终指着地面那个仍然在的偏移交叉标记,"标记位置虽然错了，但两个交叉点的直线度没问题。你手稳。"

小码没说话，但嘴角有了一点不易察觉的弧度。

他把钢筋扔在地上，重新蹲回浇筑坑边。这一次，他没有直接灌浆——先在地上用钢筋头画了一条直线。虽然还是有点歪，但他主动把那条线画在了陆终让他先看的位置。

## 二

陆终把那堵歪墙编入了新设计。不是绕过它——是利用它。

墙体厚度不对称导致的热容差异，可以被用来调节室内温差：厚的部分白天蓄热，夜间缓慢释放，帮老人扛过无图区昼夜温差大的几个关键时段。

他在地基图上标了一行注释。

小码蹲在一边看着——不懂注释里的每个字，但他知道这是在看他的产物被吸收进正式设计的入口进了库。

陆终什么都没说，但他把那面歪墙在地基图上标成了"第一基准面"。
