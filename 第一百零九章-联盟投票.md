# 第一百零九章：联盟投票

新苍穹复兴计划的投票——在陆辰回到地球的第七天举行。

不是地球内部的投票——是协议联盟的投票。新苍穹派通过某种渠道——将复兴计划的提案直接提交给了联盟核心网络——请求联盟批准在该计划的框架下——对人类文明进行强制性能力评估。

"他们绕过了地球政府。"苏晚晴从联盟联络处发回消息，"直接向联盟提交了提案。联盟的七个成员——每个都有投票权。"

"投票结果呢？"

"三天后公布。"

三天——陆辰没有闲着。他通过苍穹系统的协议层——向联盟核心网络发送了一份正式声明。声明的内容很简单：人类文明的评估——应该由人类自己决定。任何强制性评估——都需要经过地球内部的政治程序批准——不能由外部力量单方面施压。

声明发出去后——联盟核心网络的反馈——比预期更快。

"陆辰博士的声明——已被联盟核心网络记录。"哨兵的自动回复显示，"声明将被纳入投票参考材料。"

三天后——投票结果公布。

七票中——三票支持、两票反对、两票弃权。提案——未通过。

新苍穹派的复兴计划——在联盟层面被否决。但这并不代表争议结束。因为弃权的两票中——有一票附带了条件性声明："如果人类文明内部达成共识——联盟可以重新考虑此提案。"

"这不是结束。"宋知意在投票结果公布后的第一个公开声明中说，"这只是开始。"
